# Debug Command

Focused troubleshooting of a specific issue.

## Input
$ARGUMENTS

This could be:
- Description of the bug or unexpected behavior
- Error message or stack trace
- File path to investigate

## Instructions

1. **Understand the problem**
   - Parse the error message or description
   - Ask clarifying questions if the issue is unclear
   - Identify what the expected vs actual behavior is

2. **Investigate**
   - Check relevant files and trace the logic
   - Look for related code paths
   - Identify the root cause, not just symptoms

3. **Explain what's wrong**
   - Describe the root cause clearly
   - Explain why this causes the observed behavior

4. **Propose a fix**
   - Outline the solution in small, incremental steps
   - Note any potential side effects

5. **Implement the fix**
   - One step at a time
   - Verify each change before moving to the next
   - Confirm the issue is resolved
